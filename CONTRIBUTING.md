# Contributing to SpaceWeather-Hemispheric-Analysis

Thank you for your interest in contributing to **SpaceWeather-Hemispheric-Analysis**.

This project is an academic research software package designed for reproducible investigations of hemispheric geomagnetic responses during Solar Cycle 24.

## Ways to Contribute

Contributions are welcome in areas including:

* Bug fixes
* Documentation improvements
* Performance optimization
* Code refactoring
* Scientific validation
* Additional visualizations
* Support for additional observatories
* Enhanced statistical analysis

## Before Contributing

Please:

1. Read the documentation in the `docs/` directory.
2. Understand the processing workflow.
3. Preserve the existing repository structure.
4. Maintain compatibility with existing modules whenever possible.

## Coding Guidelines

* Follow the existing modular architecture.
* Keep modules focused on a single responsibility.
* Write clear, readable Python code.
* Add comments where scientific logic is not immediately obvious.
* Update documentation when changing functionality.

## Testing

Before submitting changes:

* Verify the modified module executes successfully.
* Ensure downstream modules continue to function.
* Confirm that outputs remain scientifically valid.

## Reporting Issues

If you encounter a bug, please include:

* Operating system
* Python version
* Module affected
* Error message
* Steps to reproduce the issue

## Feature Requests

Suggestions for new functionality are welcome, particularly those related to:

* Space weather analysis
* Geomagnetic data processing
* Statistical methods
* Scientific visualization
* Workflow automation

## Code of Conduct

Contributors are expected to interact respectfully and professionally.

## Acknowledgements

Thank you for helping improve **SpaceWeather-Hemispheric-Analysis** and supporting open, reproducible scientific research.
