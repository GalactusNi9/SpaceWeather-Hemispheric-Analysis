# Module 12 — 12SpaceWeatherSimulation

## Technical Module Documentation

---

# Purpose

`12SpaceWeatherSimulation.py` creates the first dynamic visualization of the processed Solar Cycle 24 datasets by animating the evolution of geomagnetic field measurements alongside key OMNI space weather indices.

The module integrates the master datasets produced during preprocessing into a synchronized animation, allowing temporal exploration of geomagnetic responses throughout the complete study period.

Unlike previous analytical modules, this module emphasizes scientific visualization rather than statistical computation.

# Scientific Operations

The module performs the following workflow.

1. Load SJG master dataset.
2. Load PIL master dataset.
3. Load OMNI master dataset.
4. Synchronize datasets.
5. Construct animation frames.
6. Update geomagnetic observations.
7. Update planetary indices.
8. Render each animation frame.
9. Export animated GIF.

# Animation Strategy

The animation advances through the master datasets using fixed frame increments.

```text
Complete Dataset

↓

50 Observations

↓

Next Frame

↓

Repeat
```

This approach substantially reduces rendering time while preserving the overall temporal evolution.

---

# Synchronization

The module aligns

* observation time,
* geomagnetic measurements,
* planetary indices

before visualization to ensure simultaneous progression across all subplots.
