# 1. Introduction

SpaceWeather-Hemispheric-Analysis is a modular scientific software package developed for analysing hemispheric geomagnetic responses using observations from the INTERMAGNET global observatory network together with hourly OMNI space weather indices.

The software forms a complete scientific workflow beginning with raw observatory measurements and ending with statistical analysis, visualization, and animated simulations of geomagnetic activity throughout Solar Cycle 24.

Unlike conventional software that attempts to automate every task in a single execution, this package intentionally separates each scientific stage into individual modules. This design allows researchers to inspect every intermediate result, validate preprocessing steps, modify parameters where necessary, and repeat only the stages affected by new datasets or methodological changes.

The software was originally developed to investigate hemispheric differences in geomagnetic responses using H-component measurements recorded at the Pilar (PIL) and San Juan (SJG) INTERMAGNET observatories together with Dst and Kp geomagnetic indices obtained from the OMNI database.

---

# 2. Scientific Background

Space weather describes the changing environmental conditions in near-Earth space caused primarily by solar activity. Coronal Mass Ejections (CMEs), high-speed solar wind streams, and solar flares inject large quantities of charged particles into Earth's magnetosphere, producing geomagnetic disturbances that can be observed globally.

The Disturbance Storm Time (Dst) index measures the strength of the magnetospheric ring current and is widely used to identify geomagnetic storms. The Kp index quantifies global geomagnetic activity using observations from multiple ground observatories.

Ground observatories measure the Earth's magnetic field components continuously. The horizontal magnetic field (H-component) is particularly sensitive to geomagnetic disturbances and provides valuable information regarding ionospheric and magnetospheric current systems.

By combining OMNI space weather indices with ground-based H-component observations, researchers can investigate how different hemispheres respond to identical solar forcing conditions.

# 24. Scientific Interpretation

Positive Pearson coefficients indicate variables that increase together.

Negative coefficients indicate inverse relationships.

Large negative Dst values correspond to stronger geomagnetic storms.

Elevated Kp values represent enhanced global geomagnetic activity.

Differences between SJG and PIL responses may indicate hemispheric asymmetries arising from seasonal, conductivity, or magnetospheric effects.
