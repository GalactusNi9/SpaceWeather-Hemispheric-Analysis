# Code of Conduct

## Our Commitment

We are committed to fostering an open, respectful, and inclusive environment for everyone interested in contributing to **SpaceWeather-Hemispheric-Analysis**.

Participation in this project should be a positive experience regardless of age, nationality, ethnicity, gender identity, disability, academic background, or level of programming experience.

---

## Expected Behaviour

Contributors are encouraged to:

* Be respectful and professional.
* Provide constructive feedback.
* Support reproducible scientific research.
* Respect differing scientific viewpoints.
* Communicate clearly and courteously.
* Credit the work of others appropriately.

---

## Unacceptable Behaviour

The following behaviours are not acceptable:

* Harassment or discrimination.
* Personal attacks or insulting language.
* Deliberate disruption of discussions.
* Misrepresentation of scientific results.
* Plagiarism or misuse of copyrighted material.
* Publishing confidential information without permission.

---

## Scientific Integrity

As this repository supports scientific research, contributors are expected to:

* Preserve the integrity of scientific analyses.
* Report errors honestly.
* Clearly document modifications.
* Avoid manipulating results to support predetermined conclusions.
* Promote reproducibility and transparency.

---

## Scope

This Code of Conduct applies to:

* GitHub Issues
* Pull Requests
* Discussions
* Documentation
* Any collaborative communication related to this repository.

---

## Enforcement

Project maintainers reserve the right to remove contributions or restrict participation where behaviour violates this Code of Conduct.

---

## Acknowledgement

By participating in this project, contributors agree to uphold these principles and help maintain a respectful and collaborative research environment.
