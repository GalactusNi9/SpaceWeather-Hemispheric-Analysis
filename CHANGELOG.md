# Changelog

All notable changes to **SpaceWeather-Hemispheric-Analysis** will be documented in this file.

The project follows a simplified form of Semantic Versioning (MAJOR.MINOR.PATCH).

---

## [1.0.0] - 2026-07-11

### Initial Public Release

This release represents the first public version of the software developed from the research project on hemispheric geomagnetic responses during Solar Cycle 24.

### Added

* Initial release of all 15 Python modules.
* Complete modular scientific processing workflow.
* H-component preprocessing utilities.
* Dst index processing and conversion.
* Geomagnetic storm detection algorithms.
* Event selection workflow.
* Multi-source dataset merging utilities.
* Pearson correlation analysis modules.
* Scientific visualization tools.
* Animation modules (AnimatorV1 and AnimatorV2).
* Scientific simulation modules (Simulecrum V1.1 and Simulecrum V1.2).
* Comprehensive project documentation.
* GitHub repository structure.
* Sample datasets.
* Sample outputs.
* Installation, User, Technical, Developer and Data Preparation guides.
* GitHub publishing documentation.
* Citation metadata.
* MIT License.

### Notes

This version reproduces the software architecture and scientific workflow used in the original research project while providing a structured, documented, and reproducible repository for future research and development.
